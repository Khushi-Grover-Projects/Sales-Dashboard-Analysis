# Sales Dashboard Analysis

## Project Overview
Created an interactive Sales Dashboard to track revenue, orders, and fulfillment rates across regions and products. Data was cleaned and transformed using Excel and Power BI for analysis.

## Dataset
- `sales_data_raw.xlsx` → Raw, uncleaned sales data
- `sales_data_cleaned.xlsx` → Cleaned and structured data

## Dashboard Components
- **KPI Cards:** Total Revenue, Total Orders, Completed Orders, Fulfillment Rate (%)  
- **Bar Chart:** Revenue by Region  
- **Column Chart:** Revenue by Product  
- **Column Chart:** Monthly Revenue Trend  
- **Donut Chart:** Revenue Share by Region  
- **Slicers:** Filter by Product and Region  

## Key Insights
- North region generates the highest revenue  
- Laptops are top-selling products  
- Fulfillment rate varies by region, indicating efficiency differences  
- Monthly trends show peaks in January and November  

## Tools & Skills
- Power BI: Dashboard visualization, KPIs, slicers, formatting  
- Power Query: Data cleaning and transformation  
- DAX: Measures for Revenue, Total Orders, Fulfillment Rate, Completed Orders
